# Головна сторінка: фото команди + розклад і таблиця (cvf.cz)

Що додає цей пакет:
- Велике фото команди на всю ширину (з підписом чеською).
- Нижче — розклад та турнірна таблиця з офіційного сайту cvf.cz (P-M-1).

## Як застосувати
1) На GitHub у вашому репозиторії натисніть **Add file → Upload files**.
2) Розпакуйте ZIP у себе і перетягніть папки **frontend/** та **frontend/public/** і файл **README-FINAL-HOMEPAGE.md** у GitHub.
3) Натисніть **Commit changes** (можна через Pull request → Merge).

## Як замінити фото
- Замініть файл `frontend/public/images/team.jpg` своїм фото (така ж назва).
- Закоміть зміну — і воно з’явиться на сайті.
